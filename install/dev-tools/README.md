# Development Tools (CLI)

This directory contains scripts for installing essential command-line tools and utilities specifically tailored for software development.

This corresponds to the **Development Tools (CLI)** category in the `SOFTWARE_INDEX.md`.

## Scripts

This directory contains the following installation scripts, mirroring the sub-categories in `SOFTWARE_INDEX.md`:

*   `tools.sh`: Installs build tools like `build-essential` and code search tools like `ripgrep`.
*   `git.sh`: Installs Git and related tools like `gh`.
*   `nodejs.sh`: Installs Node.js runtime and npm.
*   `bun.sh`: Installs Bun runtime and package manager.
*   `uv.sh`: Installs uv Python package manager.
*   `gemini-cli.sh`: Installs Google Gemini CLI.
*   `claude-code.sh`: Installs Anthropic Claude Code CLI.
*   `services.sh`: Installs Docker Engine & CLI and provides notes for VS Code Server.
